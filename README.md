# Mi Proyecto

## Descripción
Programa en Java que calcula el área, perímetro y diámetro de un círculo dado su radio.

## Lenguaje
Java

## Funciones
- `calcularArea`: Calcula el área de un círculo
- `calcularPerimetro`: Calcula el perímetro de un círculo
- `calcularDiametro`: Calcula el diámetro de un círculo

## Como ejecutar
javac funcion.java
java funcion
